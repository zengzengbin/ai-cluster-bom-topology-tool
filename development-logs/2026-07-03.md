## 已完成事项

### 1. GitHub Pages 自动部署

- 新增 `.github/workflows/pages.yml`。
- 配置 GitHub Actions 在 `main` 分支 push 或手动触发时执行部署。
- workflow 主要步骤：
  - checkout 仓库代码；
  - 安装 pnpm；
  - 使用 Node 20；
  - `actions/configure-pages@v5` 配置 Pages，并启用 `enablement: true`；
  - 执行 `pnpm install --frozen-lockfile`；
  - 执行 `pnpm build`；
  - 上传 `dist` 目录作为 Pages artifact；
  - 使用 `actions/deploy-pages@v4` 发布。
- 已通过 GitHub Actions 验证，`build` 和 `deploy` 均成功。
- 当前 Pages 地址：`https://zengzengbin.github.io/ai-cluster-bom-topology-tool/`。
